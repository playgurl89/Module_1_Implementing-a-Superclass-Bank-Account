# CSC372 Module 1 - Bank Account Inheritance

## About This Project

This project was created for CSC372-1 Programming II and serves as my first introduction to implementing inheritance in Java. The assignment challenged me to create a BankAccount superclass and extend its functionality through a CheckingAccount subclass.

While the project is relatively simple, it helped me better understand how inheritance reduces duplicated code and allows developers to build specialized classes from a common foundation.

## What the Program Does

The program allows a user to:

* Create a bank account
* Make deposits
* Process withdrawals
* Apply overdraft fees when necessary
* Display account information and balances

The CheckingAccount class inherits common account functionality from the BankAccount class while adding overdraft processing and interest rate information.

## What I Learned

The biggest takeaway from this assignment was seeing inheritance in action rather than simply reading about it. Creating a subclass that could reuse methods and attributes from a parent class made it clear why object-oriented programming is useful for building scalable applications.

I also gained more experience organizing Java projects into multiple source files and managing those files through GitHub version control.
